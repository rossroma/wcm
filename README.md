# WCM

> 灌溉收费管理系统（Watering Charge Management）



### 简介

一个基于web服务的灌溉收费管理工具，主要功能有灌溉记录新增、修改、删除，数据汇总、统计以及回收站功能。后期可能会增加导出、打印和智能纠错等功能。建议使用较新的Web标准浏览器（如Chrome，Safari，Edge等）来访问。
